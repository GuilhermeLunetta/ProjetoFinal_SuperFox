# Como jogar
 - O jogo em si já explica basicamente o que fazer, no entanto, aqui vai um tutorial:
   
   * Abra o Prompt de Comando e entre no diretório onde clonou o repositório
   * Digite: python SuperFox.py
   * O jogo irá abrir, mas não começará, você deve apertar qualquer tecla para iniciar, MAS CALMA! Leia as instruções abaixo
   * Para andar use as setas do teclado
   * Para pular use a barra de espaço
   * Para atirar aperte Q
   * Para recarregar aperte R
   * Na parte superior direita você pode ver suas bolas de fogo
   * Na parte superior central você pode ver sua pontuação
   * Você tem apenas 1 vida, use bem ela!
   * Bom jogo!
